# 上肢推拉：俯卧撑、推胸、划船与下拉

水平推动作包括俯卧撑、哑铃卧推和器械推胸；水平拉动作包括坐姿划船和单臂哑铃划船。初学者可让推与拉的总工作组大致平衡，再根据目标和耐受调整。动作选择不需要追求“唯一最佳”，稳定、无异常疼痛并能渐进最重要。

俯卧撑时，双手位置允许个体化，身体从头到脚保持整体张力，胸口向双手之间下降，再推离地面。完整地面俯卧撑过难时，应把双手放到稳定高台上做上斜俯卧撑，而不是用塌腰和缩短幅度勉强完成。随着能力提高，可逐渐降低支撑高度。

哑铃卧推时保持足部稳定，肩胛在凳面上获得支撑，前臂在主要发力阶段大体可控。肘部不必固定成完全贴身或完全外展。若肩前侧疼痛，可先减重、调整握距和动作深度，或改用中立握哑铃与器械推胸。

划船和下拉应先确定躯干稳定，再让肘部沿目标路径运动。不要为了让重量到达身体而反复耸肩、甩动或大幅后仰。下拉不是必须拉到颈后；拉向胸前通常更容易控制。

训练后的胸背肌酸胀常见，但肩关节锐痛、夜间痛不断加重、明显无力、脱位感或手臂麻木需要降低或停止相关动作，并根据严重程度接受专业评估。
